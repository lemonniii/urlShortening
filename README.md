# urlShortening
URL shortening fullstack web service

################################################################################

**Url Shortening API**

Stack: NodeJS, Express, MongoDB

Setup:
1) Navigate to url-shortening-api folder
2) Install Node if you don't have it on your computer already (from https://nodejs.org/)
3) Install dependencies
```
npm install
```
4) If you don't have mongoDB, you can use homebrew to install it on your computer. 
Then follow the instructions under https://docs.mongodb.com/getting-started/shell/tutorial/install-mongodb-on-os-x/#run-mongodb 
to be able to run mongoDB with this web service.
 ```
 brew install mongodb
 ```
 If you don't have homebrew, go to https://brew.sh/ and use the script to install homebrew on your computer first.



5) Run the API 
```
node index.js
```



################################################################################

**Url Shortening Web**

Stack: ReactJS, BootStrap, CSS

Setup: 
1) Navigate to url-shortening-web folder
2) Install dependencies
```
npm install
```
3) Start the front-facing application
```
npm start
```
